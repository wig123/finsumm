【数据关系】
- 关键数值和变化：
  - Memory OP在2018年达到峰值，约14,000 KRW bn，然后在2023年跌至最低点，约-8,000 KRW bn。
  - Memory OP margin在2018年达到最高，接近100%，随后在2023年降至约-20%。
- 极值对比：
  - DRAM OP margin在2018年高于其他年份的水平，随后的下降也非常明显。
  - NAND OP margin在2022年跌至最低，接近-80%。
- 重要趋势：
  - 09至18年，Memory OP与各项margin指标呈上升趋势，显示出显著增长。
  - 18年后均出现显著下降，尤其在21至23年期间显示“Hard-landing”现象，之后略有恢复趋势。

【模式特征】
- 全序列呈多阶段起伏，2018年前后为显著峰值，2023年前后为显著谷值，随后在24-25F出现温和修复。
- 三条margin与灰色柱体整体同向，且相对排序较稳定：DRAM高于总margin，总margin高于NAND，极端阶段三者间距扩大。
- 个别异常包括2022年NAND的极端低位，以及2020年前后的短暂低迷。

【核心洞察】
核心结论：（利润对周期与产品结构依赖度高）

业务含义：
- 盈利能力具强周期性与高弹性
- DRAM盈利性优于NAND，组合主导利润率
- 预测期仅部分恢复，难达历史峰值

风险关注：
- Memory OP margin低于0%，或持续为负，年度亏损或扩大
- NAND OP margin接近-80%，可能单项拖累合并利润至负值
