【图表构成】
- 类型：双线折线图（两条时间序列）
- 图例：US IP % y/y（浅青色）、MX IP % y/y（深绿色）
- 横轴：年份，范围2003—2023
- 纵轴：年同比百分比，范围-20%至10%，含0%基线
- 标题：未见主标题

【数据关系】
- 期初（2003）：两者均接近0%。
- 2009附近谷值：US≈-12%，MX≈-15%，差≈3个百分点。
- 2010附近峰值：US≈+8%，MX≈+6%，差≈2个百分点。
- 2020谷值：US≈-9%，MX≈-20%（或更低），差≥11个百分点。
- 2021峰值：US触及≈+10%，MX≈+6%，差≈4个百分点。
- 期末（2023）：US≈+2%～+3%，MX≈+3%～+4%。

【模式特征】  
两条时间序列在除个别年份外大部分时间方向一致、波动节奏相近，围绕0%上下频繁穿越。2008–2010与2020前后均出现同步的大幅偏离区间，但2020年深绿色线的极端偏离幅度明显更大。2010–2019阶段两线多次短期分叉后重新收敛，2023年末再次靠近。

【核心洞察】  
核心结论：两国工业增速相关性高，但2020年差异显著放大。

业务含义：  
- 两国工业周期高度联动，易形成同步波动。  
- 2020年前后墨西哥工业对冲或分散能力较弱。  
- 近期数值重新接近，显示阶段性趋同格局。

风险关注：  
- 当两者差值≥10个百分点时，跨国产能与供应链错配压力显著上升。  
- 当任一国落入≤-10%区间时，下游订单与现金流承压、违约概率上升。
